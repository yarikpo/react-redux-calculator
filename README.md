# react-redux-calculator

## Installation

```bash
git clone https://github.com/yarikpo/react-redux-calculator.git
```

```bash
cd client && npm i
```

* Install required java dependencies

## Run

* Run Java server (on port 8080)

From ```client``` folder write:
```bash
npm run start
```

* Go to http://localhost:3000
